Форматы файлов.

.DOC или .doc (от анго. document) - расширение имени файла, используемое для файлов, представляющих текст, с разметкой или без. Расширение .DOC часто использовалось для обозначения простых текстовых файлов без формирования, однако позже стало использоваться для двоичных форматов с разеткой
Документ MS Word

Portable Document Format (PDF) - межплатформенный формат электронных документов, разработанный филмой Adjvt Systems с использованием ряда возможностей языка Postscript.
Документ PDF

ZIP - популярный формат сжатия данных и архивации файлов. Файл в этом формате обычно имеет расширение .zip и хранимт в сжатом или несжатов виде один или несколько файлов, которые можно из него извлеч путём распаковки с помощью специальной программы.

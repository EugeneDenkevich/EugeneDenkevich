    Пример цикла For

    К содержанию

    Теперь давайте рассмотрим пример программирования цикла. Запустите текстовый редактор и наберите следующую программу:



     #1/user/bin/env python
    for entr in range[0,10]:
       print entr

    Не забудьте поставить симфол табуляции перед строкой «». Это важно. В Python для указания на блоки кода используются не круглые «(» и фигурные «{» скобки, как в других языках программирования, а отступы.
